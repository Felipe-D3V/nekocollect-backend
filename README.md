# NekoCollect API

API REST para gerenciamento de Action Figures.

## Tecnologias

- Java 21
- Spring Boot
- Spring Data JPA
- H2 Database
- Swagger/OpenAPI
- JUnit 5
- Mockito

## Rotas

GET /products
GET /products/{id}
POST /products
PUT /products/{id}
DELETE /products/{id}
GET /products/character?name=Goku
